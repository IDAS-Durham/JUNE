june.commute.CommuteGenerator¶

class june.commute.CommuteGenerator(regional_generators: Dict[str, june.commute.RegionalGenerator])

Generate a mode of transport that a person uses in their commute.

Modes of transport are chosen randomly, weighted by the numbers taken from census data for each given Output area.

regional_generators

A dictionary mapping Geography msoareas to objects that randomly generate modes of transport
