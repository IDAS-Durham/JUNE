class june.groups.commute.commutehub.CommuteHubs(commutecities)¶

Initialises commute hubs given the location of the commute cities they are affiliated to

Assumptions: - Each non-London station has 4 commute hubs associated with it

(this is based on the Network Rail maps showing most major stations have 4 incoming lines)

  • London has 8 commmute hubs associated with it

  • The locations of the hubs are uniformly distibuted around a circle (this means we may not exactly account for the correct mixing as we draw from different locations

    the real stations, but we believe this gives a good approximation at the sub-regional level)

commutecities: (list) members of CommuteCities msoa_coordinates (pd.Dataframe) Dataframe containing all MSOA names and their coordinates init: (bool) if True, initialise hubs, if False do this manually members: (list) list of all commute hubs
