Saves the Population object to hdf5 format file file_path. Currently for each person, the following values are stored: - id, age, sex, ethnicity, area, subgroup memberships ids, housemate ids, mode_of_transport,

population

population object

file_path

path of the saved hdf5 file

chunk_size

number of people to save at a time. Note that they have to be copied to be saved, so keep the number below 1e6.
