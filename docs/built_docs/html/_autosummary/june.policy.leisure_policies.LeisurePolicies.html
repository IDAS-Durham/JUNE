Applies leisure policies. There are currently two types of leisure policies implemented: CloseLeisureVenue, and ChangeLeisureProbability. To ensure compatibility when adding multiple policies of the same type, we “clear” the leisure module at the beginning of each application, ie, we set closed_venues = set(), and the original probabilities for each social venue distributor. We then apply the policies currently active at the given date.
