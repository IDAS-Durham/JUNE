june.groups.hospital.Hospital¶

class june.groups.hospital.Hospital(n_beds: int, n_icu_beds: int, area: str = None, coordinates: Optional[Tuple[float, float]] = None, trust_code: str = None)

The Hospital class represents a hospital and contains information about its patients and workers - the latter being the usual “people”.

We currently use three subgroups: 0 - workers (i.e. nurses, doctors, etc.), 1 - patients 2 - ICU patients

Create a Hospital given its description.

n_beds:

total number of regular beds in the hospital

n_icu_beds:

total number of ICU beds in the hospital

area:

name of the super area the hospital belongs to

coordinates:

latitude and longitude

Allocate a patient inside the hospital, in the ward, in the ICU, or transfer. To correctly log if the person has been just admitted, transfered, or released, we return a few flags: - “ward_admitted” : this person has been admitted to the ward. - “icu_admitted” : this person has been directly admitted to icu. - “ward_transferred” : this person has been transferred to ward (from icu) - “icu_transferred” : this person has been transferred to icu (from ward) - “no_change” : no change respect to last time step.
