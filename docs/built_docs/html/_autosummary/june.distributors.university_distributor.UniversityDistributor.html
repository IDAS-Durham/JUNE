june.distributors.university_distributor.UniversityDistributor¶

class june.distributors.university_distributor.UniversityDistributor(universities: List[june.groups.university.University], max_number_of_super_areas=20)

For each university it searches in the nearby super areas for students living in student households. Once it has enough to fill the university, it stops searching and fills the university.

universities

a list of universities to fill

max_number_of_super_areas
