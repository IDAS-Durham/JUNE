june.interaction.matrix_interaction¶

class june.interaction.matrix_interaction.MatrixInteraction¶

We assume that the matrices are symmetric, with age indexing rows and columns, and that they are organised in a dictionary with the group_type as key. The sum over a row then gives th total number of interactions a person of a given age has per day in this group (we can vary this number on a daily base with a Poissonian distribution) - if we call each group only once per day, this translates immediately into contacts per call. The resulting number of interactions is then distributed over group members according to the frequency in the matrix.
